# KnowledgeHub.MiniRag

A minimal Retrieval-Augmented Generation (RAG) system built with:

- ASP.NET Core
- EF Core
- SQL Server
- Qdrant
- OpenAI Embeddings

The goal of this project is to demonstrate how to build a clean architecture RAG system in .NET using your own data.

## Features

- Document ingestion
- Text chunking
- Vector embeddings
- Semantic search
- RAG-based chat

## Architecture

Clean Architecture with the following layers:

- API
- Application
- Domain
- Infrastructure

## Tech Stack

- .NET
- ASP.NET Core
- EF Core
- SQL Server
- Qdrant
- OpenAI

## Status

Work in progress 🚧

This project is being built step-by-step as a learning exercise.
