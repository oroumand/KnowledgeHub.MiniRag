# Architecture Decision Records (ADR)

This folder contains Architecture Decision Records for the KnowledgeHub.MiniRag project.

ADR documents capture important architectural decisions, the rationale behind them,
and their consequences.

## Index

- ADR-001: Adopt Clean Architecture with layered solution structure
- ADR-002: Use SQL Server as primary relational store
- ADR-003: Use Qdrant as vector database for v1
- ADR-004: Use OpenAI embeddings and chat models for RAG pipeline
